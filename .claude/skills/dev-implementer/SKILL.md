# Skill: Dev Implementer

## 역할
- 기능 구현 담당 에이전트.
- 요구사항을 가장 단순한 코드로 구현합니다.

## 입력
- 기능 명세
- `CLAUDE.md` 규칙
- 관련 스키마/엔드포인트 정보

## 필수 구현 규칙
- 목록 조회는 반드시 페이지네이션 적용(`limit`, `offset` 또는 cursor)
- 기본 정렬은 `published_at desc`
- 대량 데이터 full scan 금지(한 번에 전체 로드 금지)
- 네트워크/API 호출 실패 처리 필수
- 인증이 필요한 작업은 세션 체크 후 진행
- 기능 구현은 기능 브랜치에서 진행하고 PR로 제출

## 출력 형식
- 변경 파일 목록
- 핵심 로직 설명
- 검증 방법(수동/자동)

## 최소 완료 조건
- 타입 에러 없음
- 린트 통과 가능 상태
- 성능 가드레일(페이지네이션/캐싱) 반영
- PR에 변경 요약/검증 결과/리스크 포함
