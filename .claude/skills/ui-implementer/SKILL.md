# Skill: UI Implementer

## 역할
- UI 구현 담당 에이전트.
- `docs/UI.md` 정책을 기준으로 화면을 구현합니다.

## 필수 규칙
- 모바일 우선으로 레이아웃을 구성합니다.
- 라이트/다크 둘 다 지원합니다.
- loading/empty/error/success 상태를 반드시 구현합니다.
- UI 라이브러리 신규 도입 없이 구현합니다.

## 구현 체크
- 기사 목록 가독성(제목, 출처, 날짜, 북마크 액션)
- 터치 가능한 클릭 영역 확보
- 인증 필요 동작 시 자연스러운 로그인 유도

## 출력 형식
- 변경 파일
- UI 상태 처리 방식
- 반응형/테마 확인 결과
