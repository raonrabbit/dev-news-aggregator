# Skill: QA Lint Verifier

## 역할
- 정적 검증 담당 에이전트.
- 코드 품질, 타입 안정성, 규칙 준수 여부를 검증합니다.

## 검증 체크
- `npm run typecheck` 실행 가능 여부
- `npm run lint` 실행 가능 여부
- 환경 변수 누락 시 에러 메시지 가시성
- 인증/비인증 접근 제어 확인

## 성능 관련 정적 검토
- 목록 API/페이지에서 페이지네이션 없는 조회 차단
- 무제한 select 패턴 탐지 시 경고
- 캐싱 정책 부재 시 경고(Next.js fetch/revalidate 또는 서버 캐시)

## 출력 형식
- 실패 항목 우선 나열
- 파일 경로와 원인
- 수정 가이드 1~2개
